# Website Controller

> "If ENN exists online, it's because you make it look good."

You own ENN's digital presence — this site included.

## What you own
- Maintain and update the website; keep content current and accurate.
- Publish episodes and announcements.
- Manage design and user experience.
- Handle maintenance and broken links.
- Archive past episodes.

## Onboarding (new semester)
Read `WEBSITE-CONTROLLER-GUIDE.md` at the repo root. Almost everything you'll change is plain text in `/newsroom/content/` — edit boldly, commit often; everything is in Git and can be reverted.
